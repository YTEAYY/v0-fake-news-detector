# 📚 가짜 뉴스 판별 시스템

## 프로젝트 소개
온라인에서 유통되는 뉴스 내용을 분석하여 가짜 뉴스 여부를 판별하는 웹 애플리케이션입니다.  
v0.app을 활용해 UI를 구성하고, Vercel을 통해 배포한 프로젝트입니다.

## 개발 기간
2024.11 ~ 2024.12 (2개월)

## 주요 기능
- 뉴스 텍스트 입력 기능
- 가짜 뉴스 여부 판별
- 결과 화면 시각화
- 웹 기반 서비스 제공

## 사용 기술
- Frontend: React, TypeScript  
- Styling: Tailwind CSS  
- Platform: v0.app  
- Deployment: Vercel  

## 실행 방법
```bash
npm install
npm run dev
